
# ThingsPanel-App


## 功能特点

- 使用 Uniapp 开发，可以方便的编译成 iOS，安卓，微信小程序以及其他小程序，H5。
- 可以扫码添加设备（设备需要在后台先导入）。
- 查看监测值。
- 切换智能化业务和设备分组。
- 手动控制。
- 设置控制策略，分为设备触发和时间条件触发两种。
- 查看操作日志。
- 个人账号管理功能。
- 手机验证码登录。
